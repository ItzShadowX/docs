# Hytale Shop Setup

## Installation

1. Confirm the server or environment matches the supported release requirements.
2. Add the plugin package to the correct Hytale plugin location for your deployment model.
3. Start the environment once to generate default files if the package supports auto-generation.
4. Stop the environment before editing production configuration.
5. Keep the original package and generated defaults available for comparison during upgrades.

## First Deployment Checklist

- Confirm the runtime loads the plugin cleanly
- Store untouched generated defaults before customization
- Validate the first catalog in a staging environment
