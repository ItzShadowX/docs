# Donut Crates

Donut Crates adds crate-style rewards and key-based unlock flows for Minecraft servers. It is intended for servers that want a repeatable reward system with controlled rarity, progression, and event-based distribution.

## What This Plugin Covers

- Configurable crate definitions and reward pools
- Key-based access and distribution control
- Support for progression, events, or monetized reward loops
- Admin-friendly balancing and maintenance workflow
