# Donut Crates Configuration

Plan crate structure before editing values in production.

## Focus Areas

- Crate names and categories
- Reward tables and rarity balance
- Key acquisition methods
- Broadcast or animation settings
- Economy or permission integration

## Configuration Guidance

Use conservative reward values in the first rollout, then tune balance after observing player behavior. Keep reward definitions small and understandable until you have real production data on open rates and perceived value.
