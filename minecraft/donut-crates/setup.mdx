# Donut Crates Setup

## Installation

1. Confirm compatibility with your server software and release version.
2. Stop the server before installing the plugin.
3. Move the Donut Crates JAR into the `plugins` directory.
4. Start the server to generate default crate files and plugin data.
5. Stop the server and edit the generated files before going live.

## First Deployment Checklist

- Confirm the server boots cleanly with the new plugin installed
- Save the freshly generated defaults before tuning rewards
- Test on staging before importing a large production crate catalog
