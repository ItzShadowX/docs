# Donut Shops Operations

## Admin Usage

1. Test purchases with a staff account and a normal player account.
2. Confirm balance deduction, item delivery, and failure messaging.
3. Review behavior when inventories are full or transactions are invalid.
4. Validate that restricted categories stay hidden or blocked as expected.
5. Announce pricing changes carefully when updating a live economy.

## Troubleshooting

### Purchases fail or do nothing

Inspect:

- Economy plugin status
- Pricing configuration validity
- Inventory delivery constraints
- Permission checks

### Shop menus look inconsistent

Review formatting, category order, and any custom text fields introduced during setup.

### Live pricing changes caused issues

Revert to the last known-good configuration backup and reintroduce changes in smaller batches.
