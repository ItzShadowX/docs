# Donut Shops Configuration

Structure the shop around clear categories and predictable player expectations.

## Review Areas

- Product categories and listing order
- Buy and sell pricing
- Economy provider integration
- Display names, descriptions, and messages
- Access restrictions based on permissions or progression

## Configuration Guidance

Avoid shipping a large catalog on day one. Start with a narrow set of listings, then expand after verifying economic balance and player behavior.
