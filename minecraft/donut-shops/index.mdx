# Donut Shops

Donut Shops provides an in-game shop system for Minecraft servers. It is intended for creators who need a maintainable way to define product listings, pricing, and player purchase flow without relying on ad hoc admin processes.

## What This Plugin Covers

- Configurable shop listings and categories
- Player purchase flow inside the server experience
- Pricing controls for stable server economies
- Clear admin maintenance workflow for catalog changes
