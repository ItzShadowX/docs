# Donut Shops Setup

## Installation

1. Verify the plugin release matches your server environment.
2. Stop the server and place the plugin JAR in the `plugins` directory.
3. Start the server once to generate default shop files.
4. Stop the server before editing product definitions and prices.
5. Back up the generated configuration before major catalog edits.

## First Deployment Checklist

- Validate startup logs before exposing the shop to players
- Keep a backup of the generated catalog defaults
- Begin with a small product set rather than a full production catalog
