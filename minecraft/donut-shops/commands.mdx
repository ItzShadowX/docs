# Donut Shops Commands

Root aliases: `/shop`, `/donutshop`, `/donutshops`

## Player Command

| Command | Permission | Description |
| --- | --- | --- |
| `/shop` | None documented | Opens the main shop menu for the executing player. Player-only because the default action opens a GUI. Example: `/shop`. |

## Admin Commands

| Command | Permission | Description |
| --- | --- | --- |
| `/shop open <player>` | `donutshops.command.open` | Opens the main shop menu for a target player. Can be run by player or console because it accepts a `CommandSender`. Example: `/shop open ShadowX`. |
| `/shop reload` | `donutshops.command.reload` | Reloads the plugin configuration and menus. Can be run by player or console. Example: `/shop reload`. Appropriate for menu and category edits; use a restart if you changed platform or dependency state. |

## Permissions

Declared in `plugin.yml` with `default: op`:

- `donutshops.command.open`
- `donutshops.command.reload`

## Operational Notes

- The default `/shop` action is player-only because it opens a GUI.
