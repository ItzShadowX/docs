# Donut Orders Setup

## Installation

1. Confirm the server is running the Minecraft platform and version supported by the product package.
2. Stop the server before adding or replacing the plugin.
3. Place the Donut Orders plugin JAR into the server `plugins` directory.
4. Start the server once so the plugin can generate its default files and folders.
5. Stop the server again before making production configuration changes.

## First Deployment Checklist

- Keep the distributed package for reference during setup
- Verify startup logs for dependency or compatibility issues
- Generate fresh default files before copying older configuration forward
- Back up the generated defaults before the first production edit
